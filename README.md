# user-management-system-in-springboot
SpringBoot를 이용한 사용자 관리 시스템 REST API 서버입니다.

## 사용 기술

IDE : VSCode   
서버 언어 : Java   
프레임워크 : SpringBoot   
빌드도구 : Maven   
데이터베이스 : H2 Database(개발용), PostgreSQL(운영용)   
ORM : JPA, Spring Data JPA   
라이브러리: lombok   
   
화면 : HTML, CSS, SCSS   
클라이언트 언어 : JavaScript   
버전 관리 도구 : Git, GitHub   
   
테스트 도구 : Postman
